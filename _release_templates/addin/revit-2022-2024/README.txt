ExportIFCFromRevit-CSharp — Revit add-in package for Revit 2022–2024

Что это:
Этот пакет содержит Revit add-in для batch-экспорта IFC
в Revit 2022, 2023 и 2024.

Что важно понимать:
Это не standalone-приложение и не основной запускной пакет.
Сам по себе этот архив не выполняет экспорт.
Он только устанавливает add-in, который затем используется
orchestrator.

Как использовать:
1. Убедитесь, что на машине действительно используется
   Revit 2022–2024.
2. Откройте свойства файла install-addin.ps1. На вкладке «Общие»,
   если Windows показывает флажок «Разблокировать», установите его и примените изменения.
3. После этого попробуйте запустить `install-addin.ps1`
   обычным способом.
4. Если скрипт не запустился или окно сразу закрылось, откройте
   PowerShell в папке пакета и выполните:

   powershell -ExecutionPolicy Bypass -NoProfile -File .\install-addin.ps1

5. После установки один раз вручную откройте нужную версию Revit и,
   если появится окно подтверждения загрузки add-in, нажмите
   «Загружать всегда».
6. После этого используйте orchestrator-пакет для самого
   запуска экспорта.

Важно:
- этот пакет нужен только для Revit 2022–2024;
- для Revit 2025–2026 нужен другой add-in package;
- для batch-экспорта одного add-in недостаточно:
  нужен ещё orchestrator.

Где читать подробности:
- в orchestrator-пакете сначала откройте `README.txt`;
- если нужен более подробный практический маршрут,
  откройте там же `_docs/ExportIFC_manual_ru.docx`;
- полный Markdown-manual находится в репозитории проекта
  `ExportIFCFromRevit-CSharp` в файле `_docs/ExportIFC_manual.md`.
